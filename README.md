## setup project

    step 1: git clone project from github
    step 2: open project with vscode
    step 3: run terminal: "npm i" at project.
        (install  packages for project)
    step 4: create .env file and add your api key.
        api key generated from the acc you use with google AI studio

        Create file: ./database/db/ai_responses.txt

### Run project

    run terminal: "npm run dev" or "npm run start"
